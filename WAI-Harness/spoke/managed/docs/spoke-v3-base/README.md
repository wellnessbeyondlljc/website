# Spoke

Public spoke-side framework assets.

This tree is for assets that help create or define a spoke, not for a live spoke.
Live `WAI-Spoke/` state belongs in an actual spoke project created from the framework.

Current contents:

- `codebase/templates/spoke/` for spoke templates
- `codebase/templates/commands/` for command templates consumed by spokes
- `teachings/` reserved for spoke-specific teachings as they are separated from shared teachings
